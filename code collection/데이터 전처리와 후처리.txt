#빅데이터 전처리, 후처리에 관한 짤강

1. 빅데이터 전처리는 필터링, 유형변환, 정제이다. 필터링은 데이터 활용목적에 맞지 않는 정보를 제거하여 분석시간을 단축하고 저장공간을 효율적으로 사용하고, 데이터 마이닝을 통해 오류나 중복을 제거하는 과정을 말한다. 다음, 유형변환은 데이터 분석이 용이하여 데이터 유형을 변환하는 과정을 말한다. 정제는 수집된 데이터의 불일치성을 교정하기 위한 과정으로 결측치와 데이터 속의 노이즈를 제거하는 과정이다.

2. 빅데이터 후처리는 변환, 통합, 축소를 말한다. 변환이란 다양한 형식으로 수집된 데이터를 분석에 용이하도록 일관성 있는 형식으로 변환하는 것을 말한다. 일반화, 정규화 등을 거치게 된다. 데이터 통합은 상호 연관성이 있는 데이터를 하나로 결합하는 기술이다. 데이터 축소는 불필요한 데이터를 축소하여 고유한 특성은 손상되지 않도록 하고 분석에 대한 효율성을 높이는 과정이다.
